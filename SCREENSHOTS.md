# Screenshot Guide

## How to add screenshots to your README:

### 1. Take Screenshots
- **Signup Page**: Visit http://localhost:3000 and click "Sign up" to screenshot the registration form
- **Login Page**: Visit http://localhost:3000 and take a screenshot of the login form
- **Dashboard**: Login and screenshot the main dashboard with task statistics
- **Tasks View**: Screenshot the tasks list showing different task statuses and operations
- **Task Creation**: Open the "Add New Task" modal and screenshot it
- **Mobile View**: Use browser dev tools to simulate mobile and screenshot

### 2. Save Screenshots
Save the screenshots in the `images/` folder with these exact names:
- `signup-page.png`
- `login-page.png`
- `dashboard.png`
- `tasks-view.png`
- `task-creation.png`
- `mobile-view.png`

### 3. Image Size Recommendations
- **Width**: 800-1200px for desktop screenshots
- **Format**: PNG for UI screenshots (better quality)
- **File Size**: Keep under 1MB per image

### 4. Alternative: GitHub Upload
1. Go to GitHub Issues in your repository
2. Drag and drop images to upload them
3. Copy the generated URLs
4. Use those URLs in your README

## Example Markdown Syntax:
```markdown
![Alt Text](./images/screenshot.png)
![Screenshot with caption](./images/demo.png)
*Caption describing the screenshot*
```
